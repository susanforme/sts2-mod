# Jhin MVP 文档 - 阶段 13：完整遗物与药水

## 1. 阶段目标

本阶段目标是补齐烬角色的专属外围系统，包括剩余遗物与药水内容。

本阶段重点不是单纯增加可获取物件数量，而是让遗物和药水真正支撑多条 Build 路线，强化角色的构筑深度、战斗节奏变化和临时爆发选择。

## 2. 本阶段交付内容

必须实现以下遗物与药水。

## 2.1 遗物

1. `第四颗子弹`
2. `精致枪油`
3. `莲华装置`
4. `演出门票`
5. `第四幕剧本`
6. `完美舞台`
7. `禁忌改造`

## 2.2 药水

1. `装填药剂`
2. `莲华药剂`
3. `谢幕药剂`

## 3. 功能目标

本阶段内容完成后，必须实现以下目标：

- 遗物能强化不同 Build，而不是只强化华彩爆发。
- 药水能作为临时资源，提供爆发、节奏重置或解场能力。
- `禁忌改造` 作为 Boss 遗物，能够明显改变弹匣节奏，但要保留可感知代价。

## 4. 遗物设计定位要求

本阶段遗物应尽量覆盖不同玩法方向，而不是全部围绕同一种收益。

建议覆盖方向：

- 第四枪爆发
- 装填节奏
- 标记击杀
- 莲华陷阱铺场
- 回合内四张牌节奏
- Boss 战节奏改写

## 5. 各遗物需求

## 5.1 第四颗子弹

定位：

- 第四枪爆发向普通遗物。

需要完成：

- 每场战斗第一次华彩造成的伤害翻倍。

最低要求：

- 只影响每场战斗第一次华彩。
- 只影响该次华彩伤害，不影响后续华彩。
- 不应错误作用于非攻击或非华彩效果。

## 5.2 精致枪油

定位：

- 装填后射击强化遗物。

需要完成：

- 每当你装填，下一张射击牌造成的伤害 `+4`。

最低要求：

- 只作用于装填后的下一张射击牌。
- 触发后应正确清除。
- 不应无限叠加到多张后续射击。

## 5.3 莲华装置

定位：

- 莲华陷阱向罕见遗物。

需要完成：

- 每场战斗开始时，给予随机敌人 `2` 层莲华陷阱。

最低要求：

- 起手可稳定提供陷阱铺垫。
- 多敌战中随机目标必须明确。
- 与后续陷阱叠层和触发逻辑兼容。

## 5.4 演出门票

定位：

- 标记击杀收益遗物。

需要完成：

- 每当你击杀带有标记的敌人，获得 `6` 金币。

最低要求：

- 只在击杀目标死亡时仍满足“带有标记”条件才触发。
- 不能对无标记击杀错误生效。
- 不能因多段结算重复发放金币。

## 5.5 第四幕剧本

定位：

- 华彩资源回报向稀有遗物。

需要完成：

- 每回合第一次触发华彩时，抽 `1` 张牌并获得 `1` 点能量。

最低要求：

- 每回合只触发一次。
- 能稳定响应华彩事件。
- 与 `华彩节奏` 等牌叠加时不能造成明显循环失控。

## 5.6 完美舞台

定位：

- 四张牌节奏向稀有遗物。

需要完成：

- 如果你在一回合内刚好使用 `4` 张牌，回合结束时获得 `12` 点格挡。

最低要求：

- 必须是“刚好 4 张牌”，而不是至少 4 张牌。
- 回合结束时结算。
- 不应被第 5 张、第 6 张牌错误触发。

## 5.7 禁忌改造

定位：

- 改写节奏的 Boss 遗物。

需要完成：

- 子弹上限变为 `5`。
- 每第 `5` 发射击触发华彩。
- 华彩伤害提高 `75%`，而不是 `50%`。

最低要求：

- 必须真实改变弹匣节奏，不再是默认第四枪节奏。
- 必须与现有子弹、华彩、UI 逻辑兼容。
- 必须让玩家感受到收益与代价同时存在。

风险要求：

- 代价不一定是显式扣资源，但至少要体现在“破坏原本四拍节奏、影响已有构筑熟悉度”。
- 不应成为无脑最优 Boss 遗物。

## 6. 各药水需求

## 6.1 装填药剂

定位：

- 基础节奏重置药水。

需要完成：

- 将子弹恢复至 `4`。
- 抽 `1` 张牌。

最低要求：

- 能提供一次即时续航与节奏修正。
- 不应绕开已有装填逻辑产生异常状态。

## 6.2 莲华药剂

定位：

- 多敌战铺场药水。

需要完成：

- 给予所有敌人 `2` 层莲华陷阱。

最低要求：

- 全体施加逻辑稳定。
- 可明显提升多敌战解场能力。

## 6.3 谢幕药剂

定位：

- 临时终结药水。

需要完成：

- 将 `1` 张 `谢幕` 加入手牌。
- 本回合其费用为 `0`。
- 消耗。

最低要求：

- 生成的 `谢幕` 只服务当前回合。
- 临时牌与正常 `谢幕` 逻辑兼容。
- 应能提供一轮明显的临时爆发或残局解场能力。

## 7. Build 支撑要求

本阶段遗物和药水必须覆盖不同路线，而不是全部强化华彩爆发。

至少应支撑：

- 第四枪爆发：`第四颗子弹`、`第四幕剧本`
- 装填节奏：`精致枪油`、`装填药剂`
- 标记收益：`演出门票`
- 莲华陷阱：`莲华装置`、`莲华药剂`
- 四拍节奏：`完美舞台`
- 节奏改写：`禁忌改造`
- 终结能力：`谢幕药剂`

## 8. 建议实现的类与入口

本阶段建议优先完善以下类或接口：

- `FourthBulletRelic`
- `FineGunOilRelic`
- `LotusDeviceRelic`
- `ShowTicketRelic`
- `ActFourScriptRelic`
- `PerfectStageRelic`
- `ForbiddenModificationRelic`
- `ReloadPotion`
- `LotusPotion`
- `CurtainCallPotion`

以及以下系统入口：

- 战斗开始遗物触发入口
- 华彩监听入口
- 装填监听入口
- 子弹上限修改入口
- 回合结束牌数统计入口
- 临时卡加入手牌入口

## 9. 重点平衡观察项

本阶段必须持续关注以下问题：

1. 遗物是否只会继续放大华彩，而没有支撑其他 Build。
2. `第四幕剧本` 与 `华彩节奏` 是否叠加过强。
3. `精致枪油` 是否让装填收益过于稳定高效。
4. `演出门票` 是否让标记击杀在经济上过强。
5. `禁忌改造` 是否过强到压过其他 Boss 遗物选择。
6. `谢幕药剂` 是否在关键战中过于稳定地提供无代价斩杀。

## 10. 建议实现顺序

建议按以下顺序推进：

1. 先完成与已有系统最紧密的 `第四颗子弹`、`精致枪油`、`第四幕剧本`。
2. 再完成 `莲华装置` 与 `演出门票`。
3. 再完成 `完美舞台`，验证牌数节奏统计。
4. 再完成 `禁忌改造`，专项回归其对弹匣系统和 UI 的改写。
5. 最后补 3 个药水，并回归多敌战、Boss 战和残局表现。

## 11. 非目标

以下内容不属于本阶段：

- 更高级的遗物美术特效
- 非设计文档定义的新增遗物或药水
- 灵魂卡来源扩展
- 最终全量平衡定稿
- 单元测试

说明：

- 本阶段重点是完整外围系统功能可用并支撑不同 Build。

## 12. 验收标准

本阶段完成后，必须满足以下验收条件：

1. 遗物能强化不同 Build，而不是只强化华彩爆发。
2. Boss 遗物 `禁忌改造` 改变节奏但有明确代价。
3. 药水能提供临时爆发或解场能力。

建议补充手动验证场景：

1. 使用 `第四颗子弹` 时，首个华彩应有明显爆发提升。
2. 使用 `精致枪油` 后，装填后的下一张射击应获得一次性增伤。
3. 使用 `莲华装置` 与 `莲华药剂` 时，多敌战开场与中段都应有清晰铺场收益。
4. 使用 `演出门票` 时，带标记击杀应正确给金币。
5. 使用 `完美舞台` 时，仅在一回合恰好打出 4 张牌时回合结束获得格挡。
6. 使用 `禁忌改造` 后，弹匣节奏应变为 5 发一轮，且 UI、华彩、射击逻辑都保持正确。
7. 使用 `谢幕药剂` 时，应能获得一张当回合 0 费的临时 `谢幕`，并用于爆发或残局清场。

## 13. 完成定义

当以下条件全部满足时，可判定“阶段 13 完成”：

- 7 个遗物和 3 个药水已全部可用。
- 它们分别支撑了不同 Build 的外围强化需求。
- `禁忌改造` 已稳定改写节奏且未破坏基础系统。
- 药水已能在关键回合提供明确的临时价值。
- 外围系统不再明显偏向单一华彩爆发路线。

## 14. 下一阶段接口预留

为了减少后续返工，本阶段应预留以下扩展点：

- 遗物与药水平衡热修正入口
- 更复杂的 Build 统计与来源记录入口
- Boss 遗物对 UI 与机制改写的统一扩展点
- 临时牌、临时状态、临时爆发资源的通用管理入口

这部分只要求结构可扩展，不要求本阶段实现完整内容。
